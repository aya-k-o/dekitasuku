# できたすく

> 毎日の小さな頑張りを、家族の宝物に

## アプリ概要

子どもの成長・親子のコミュニケーション・振り返りの習慣を一つにまとめたアプリです。
継続して使ってもらうことを最優先に設計しています。

## 機能

- 子ども選択画面
- タスク一覧・完了ボタン・ポイント加算
- 日記・体調記録
- 親の管理画面（タスク管理・子ども管理・報酬管理・日記確認）

## 技術スタック

- PHP / MySQL / HTML / CSS / JavaScript
- Docker / phpMyAdmin
- デプロイ予定：レンタルサーバー
